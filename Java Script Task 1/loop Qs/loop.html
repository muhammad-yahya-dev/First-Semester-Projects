JavaScript Looping Challenges

1. Calculate Factorial (For Loop)

Finds $n!$ using a `for` loop.

2. Triangular Number Pattern (1, 3, 6, 10...)

Generates triangular numbers (sum of integers) using a `for` loop.

3. Print String Characters

Prints each character of the input string using a `for...of` loop.

4. Numbers Divisible by 3 (1-30)

Finds numbers divisible by 3 using a `for` loop and the modulus operator (`%`).

5. Sum of Even Numbers (1-100)

Calculates the sum of all even numbers from 1 to 100 using a `for` loop.

6. Display Squares (1 to 10)

Displays the square of each number using a `for` loop.

7. Print Even Numbers (1-50)

Prints even numbers between 1 and 50 using a `for` loop.

8. Print Odd Numbers (1-30)

Prints odd numbers between 1 and 30 using a `for` loop.

9. Count Vowels

Counts the number of vowels in a string using a `for` loop.

10. Password Validation (While & Do-While)

Demonstrates password validation using `while` and `do-while` loops.
